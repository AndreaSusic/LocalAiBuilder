1.  In **fetchGbpProducts.ts** return [] if nothing found.

2.  New client util  **src/utils/renderProducts.js**
    ```js
    export function renderProducts(products){
      if(!products?.length) return;           // nothing to do

      const grid = document.querySelector('.services-grid.three-columns');
      if(!grid) return;

      grid.innerHTML = '';                    // clear placeholders
      products.forEach(p => {
        grid.insertAdjacentHTML('beforeend', `
          <div class="service-card">
            <img src="${p.image}" alt="${p.name}">
            <h4>${p.name}</h4>
            <p>${p.description || ''}</p>
          </div>
        `);
      });
    }
    ```

3.  Add a small loader in **src/main.jsx** (or appropriate file)
    ```js
    import { renderProducts } from './utils/renderProducts';

    fetch('/api/gbp-products?cid=' + gbpCid)
      .then(r => r.json())
      .then(renderProducts)
      .catch(console.error);
    ```

   *Only call this if `bootstrap.gbpCid` exists.*

4.  Server route already exists (`/api/gbp-products`).  
    No change except return `[]` when scrape empty.
